Como funciona o contexto

Os arquivos de contexto são utilizados pelo assistant para fornecer respostas mais precisas e específicas ao seu negócio. Quando um usuário faz uma pergunta, o sistema:

  1. Analisa a pergunta do usuário
  2. Busca informações relevantes nos arquivos de contexto ativos
  3. Combina essas informações com o conhecimento do modelo de IA
  4. Gera uma resposta personalizada e precisa
